# SolvNet® Application Notes

Additional TCAD Sentaurus™ application notes are available for downloading from the Synopsys FTP server. A complete list of available application notes, with brief descriptions, is available from the SolvNet support site.

TCAD Sentaurus application notes are added continually to SolvNet for the current release. Check SolvNet regularly for updates.

# Accessing Application Notes From SolvNet

To access the application notes from SolvNet:

1. Click the link https://solvnet.synopsys.com/retrieve/026036.html.   
2. Enter your SolvNet user name and password, and click Sign In.

The List of Available TCAD Sentaurus Application Examples and Notes is displayed. This presents the latest index of application notes.

# Accessing the FTP Server Directly

To access the FTP server to download the Sentaurus Workbench simulation project files (gzp) for the application notes, for a given release:

1. Click one of the following links: https://solvnet.synopsys.com/redauthftp/cafe/TCAD\_Sentaurus\_Applications\_O-2018.06 https://solvnet.synopsys.com/redauthftp/cafe/TCAD\_Sentaurus\_Applications\_N-2017.09 https://solvnet.synopsys.com/redauthftp/cafe/TCAD\_Sentaurus\_Applications\_M-2016.12

NOTE The URLs for these links might change. Contact TCAD Support if these links do not work.

2. Enter your SolvNet user name and password, and click Sign In.   
3. Reenter your password, and click Submit or Submit Query.   
4. In the FTP window, right-click the gzp file required and choose Save Link As.   
5. Download the file to a directory below your TCAD Sentaurus root directory (STDB).   
6. Refresh the Sentaurus Workbench projects browser, and open the downloaded project.

To access the PDF documentation in Sentaurus Workbench:

1. Open the project.   
2. Choose Project > Properties > Documentation.
